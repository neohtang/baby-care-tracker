# TabBar 图标说明

此目录用于存放 TabBar 的导航图标文件。

## 需要的图标文件

| 文件名 | 用途 | 建议尺寸 |
|--------|------|----------|
| home.png | 首页（未选中） | 81x81 px |
| home-active.png | 首页（选中） | 81x81 px |
| record.png | 记录（未选中） | 81x81 px |
| record-active.png | 记录（选中） | 81x81 px |
| vaccine.png | 疫苗（未选中） | 81x81 px |
| vaccine-active.png | 疫苗（选中） | 81x81 px |
| settings.png | 设置（未选中） | 81x81 px |
| settings-active.png | 设置（选中） | 81x81 px |

## 颜色规范

- 未选中状态：#9CA3AF
- 选中状态：#7C6FE0

## 注意事项

- 微信小程序 TabBar 图标建议使用 81x81 px 的 PNG 格式
- 图标文件大小不超过 40KB
- 推荐使用 [IconFont](https://www.iconfont.cn/) 或 [Remix Icon](https://remixicon.com/) 导出合适的图标
